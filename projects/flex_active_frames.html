Flex Active Frames

Steelcase

2020

     modular storage system

     metal · wood

     felted PET · injected plastic

Steelcase Flex Active Frames are adaptable structures that define space by creating productive, flexible and inspiring homes for teams.

The sytsem logic is designed to freely configure, add or subtract components within a defined grid. The basic grid is set up with the simple use of an Allan key. All extra components, side covers, doors and whiteboards can be assembled tool free.
